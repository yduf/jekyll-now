## Installation

### Prerequesite
- Install UI desktop on raspbian
{% highlight cpp %}
sudo /home/pi/scripts/install-desktop
{% endhighlight %}

### Driver installation
{% highlight cpp %}
git clone https://github.com/goodtft/LCD-show.git
chmod -R 755 LCD-show
cd LCD-show/
sudo ./MHS35-show
{% endhighlight %}
