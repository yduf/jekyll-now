## [using cat](https://unix.stackexchange.com/questions/144172/full-dd-copy-from-hdd-to-hdd#)

{% highlight cpp %}
cat /dev/sdb >/dev/sdc
{% endhighlight %}

## using [pipe viewer (pv)](http://man7.org/linux/man-pages/man1/pv.1.html)

{% highlight cpp %}
pv /dev/sdb > /dev/sdc
{% endhighlight %}
