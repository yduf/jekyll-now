## [SQlite online](http://tpcg.io/AT8Not )

- sum(X) filter null
- null + X => gives null

{% highlight cpp %}
select sum(val), count(*)
from ( select 1 + null as val, 1 as row
union
select 4 as val, 1 as row
) t
group by row
{% endhighlight %}
